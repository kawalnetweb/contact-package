# Contact Form laravel Package

## This will send email to admin & save data into database

1. Step 1
```
 composer require netweb/contact
```
2. Step 2 - Publish
```
 php artisan vendor:publish --tag="contact-config"

```
3. Step 3 
```
php artisan serve

```
4. Step 4 
```
    http://127.0.0.1:8000/contact
```
